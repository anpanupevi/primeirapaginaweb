INTRODUÇÃO AO HTML

HTML (HyperText Markup Language) é a "linguagem de marcação" usada para criar páginas da web. Pense nele como o esqueleto de um site, ou seja é a estrutura da pagina.

O QUE SÃO TAGS

As tags são como “comandos” para estruturar a página HTML, dizem ao navegador o que mostrar na página. Sempre começam com e terminam com , mas tem algumas tags que não presisam de ser fechadas como o Input e outras que abrem e fecham sozinhas como img. Temos muitos exemplos de tags, vide indice

Exemplos de Tags

  • <b>: deixa o texto em negrito;
  • <i>: deixa o texto em itálico;
  • <u>: sublinha o texto;
  • <mark>: marca o texto como se fosse com marca-texto;
  • <sup>: texto sobrescrito (tipo potência: 22);
  • <sub>: texto subscrito (tipo fórmulas químicas: H2O);
  • <strong>: é parecido com <b>, mas também dá importância ao texto, carga semântica;
  • <small>: deixa o texto menor;
  • <blockquote>: mostra uma citação longa;
  • <q>: mostra uma citação curta, com aspas;
  • <hr>: cria uma linha horizontal;
  • <img>: mostra uma imagem;
  • <video>: coloca um vídeo;
  • <track>: serve para colocar legendas no vídeo;
  • <iframe>: mostra outra página dentro da sua;
  • <figure>: agrupa uma imagem e sua legenda;
  • <figcaption>: legenda de uma figura;
  • <picture>: mostra imagens que mudam de acordo com o tamanho da tela.

ORGANIZAÇÃO DE CONTEÚDO

Classes: (class="...") São como "etiquetas" que você coloca em vários elementos pra estilizar ou controlar vários de uma vez (ex: .destaque { color: red; }).

ID (id="...") É um identificador único (só pode um por página). Usamos pra links internos ou JavaScript (ex: <div> id="menu">).

<div>: É um "bloco" genérico pra agrupar conteúdo ou criar seções. Não faz nada visual sozinho, mas ajuda a organizar.

<span>: Igual ao <div>, mas inline (não quebra a linha). Usamos pra estilizar pedaços de texto (ex: texto).

Âncoras (links internos): Link que leva pra outra parte da página.

LISTAS

Listas Ordenadas "<ol>": São listas numeradas, por ex: 1 Arroz; 2 Feijão; 3 Macarrão...

Não ordenadas "<ul>": São listas que são organizadas através de marcadores como bolinhas como aquelas usadas no word.

LINKS

Para criar link utilizamos a tag <a>, com essa tag, conseguimos linkar conteúdos da mesma ou de outras páginas

Ex: <a href="url"> - neste caso url será o endereço da página ou link.
Temos duas opção para criar o link, podemos usar os atributos:

* target="_blank" - Abre o link em outra aba/janela

* target="_self" - Abre o link na mesma aba (é o comportamento padrão).

FORMULÁRIOS

<form> → Cria o formulário.

<input> → Campo para digitar.

<input type="checkbox"> → Caixinha de marcar.

<input type="radio"> → Botão de opção única.

<button> → Botão clicável.

<select> → Caixa de seleção (dropdown: "opções de escolha").

<textarea> → Área de texto grande.

<label> → Rótulo do campo.

<fieldset> → Agrupa campos.

<button type="submit"> → Botão de enviar formulário.

(Voltar ao inicio)

TABELAS

<table> → Começa a tabela.

<tr> → Cria uma linha.

<td> → Cria uma coluna.

<th> → Célula de título (cabeçalho).

<thead> → Cabeçalho da tabela.

<tbody> → Corpo da tabela.

<tfoot> → Rodapé da tabela.

<caption> → Legenda da tabela.

colspan → Mescla colunas (ex: <td colspan="2">).

rowspan → Mescla linhas (ex: <td rowspan="2">).

Atributos: border, cellpadding, cellspacing, align.

MÍDIA (IMAGEM E VÍDEO)

<img> → Mostra imagens (use src para o caminho e alt para texto alternativo).

<video> → Mostra vídeos (adicione controls para player).

<track> → Adiciona legenda ao vídeo (ex: legendas .vtt).

<iframe> → Mostra outro site/página dentro da sua.

<embed> → Incorpora conteúdo externo (PDFs, plugins).

<figure> e <figcaption> → Imagem com legenda semântica.

<picture> → Imagem responsiva (muda conforme tamanho da tela).

HTML 5 & NOVIDADES

No HTML5 vieram novas tags semânticas, que ajudam a organizar o conteúdo de forma mais clara:

<main> → Conteúdo principal da página (deve ser único).

<header> → Cabeçalho da página ou de seções.

<footer> → Rodapé da página ou de seções.

<nav> → Área de navegação (menu de links).

<section> → Seção lógica do conteúdo.

<aside> → Conteúdo complementar/lateral.

<article> → Conteúdo independente (post, notícia).

ACESSIBILIDADE (WAI - ARIA)

São atributos que ajudam deficientes visuais com leitores de tela. Por exemplo:

☺ ► aria-hidden="true" → esconde do leitor de tela

☺ ► aria-expanded="false" → diz que está fechado

☺ ► aria-expanded="true" → diz que está aberto

WEB E ROBÔS

WEB SCRAPING E CRAWLER

Web scraping: é pegar informações de sites automaticamente (tipo um robô coletando dados).

Crawler: é o robô que visita páginas e indexa para os buscadores, tipo o Google Bot.

(Voltar ao inicio)

SEO (Otimização para mecanismos de busca)

É o que fazemos pra o Google encontrar e mostrar bem nosso site na rede web.

SEO: melhorar o site pra aparecer melhor nas buscas.

Google Bot: o robô do Google que visita sites.

Search Console: ferramenta do Google pra analisar o site.

Google Ads: anúncios pagos.

Schema.org: marcações no código que ajudam os mecanismos de busca a entender o conteúdo.
